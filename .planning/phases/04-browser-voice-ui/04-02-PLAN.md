    **1. Create frontend/src/hooks/useAudioVisualization.js:**

    Custom hook `useAudioVisualization(amplitude)` that smooths amplitude values for animation:
    - Takes raw amplitude (0-1) as input
    - Uses useRef for animation frame ID and smoothed value
    - Uses requestAnimationFrame loop to interpolate smoothedAmplitude toward target amplitude (lerp factor ~0.15)
    - Returns `smoothedAmplitude` (number 0-1) as state
    - Cleanup: cancelAnimationFrame on unmount using a `running` ref flag
    - This drives the pulsing circle scale: scale = 1 + smoothedAmplitude * 0.4 (so circles grow up to 40% larger)

    **2. Create frontend/src/components/AudioVisualizer.jsx:**
