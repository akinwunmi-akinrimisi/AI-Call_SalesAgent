Replace the stub in `scripts/seed_firestore.py` with a complete implementation:

1. **Imports:** os, subprocess, sys from stdlib; pymupdf4llm; google.cloud.firestore; dotenv.load_dotenv; pathlib.Path

2. **Constants:**
   - GCP_PROJECT from env or default "vision-gridai"
   - CREDENTIALS_PATH from env GOOGLE_APPLICATION_CREDENTIALS or default "secrets/openclaw-key-google.json"
   - KB_DIR = Path("knowledge-base")
   - PDF_MAP dict mapping document IDs to PDF filenames (5 entries per CONTEXT.md)
   - KEYWORDS dict mapping document IDs to manually curated keyword lists:
     - "programmes": ["cloud security", "SRE", "platform engineering", "pricing", "curriculum", "duration", "internship"]
     - "conversation-sequence": ["qualification", "opening", "recommendation", "objection", "closing", "decision tree"]
     - "faqs": ["frequently asked questions", "schedule", "prerequisites", "certification", "support"]
     - "payment-details": ["bank transfer", "payment", "account details", "installment", "pricing"]
     - "objection-handling": ["price objection", "time commitment", "job outcomes", "beginner concerns", "ROI"]

3. **ensure_firestore_db() function:**
   - Run `gcloud firestore databases create --location=europe-west1 --type=firestore-native --project={GCP_PROJECT}` via subprocess.run
   - Capture output, check returncode
   - Print [OK] for success or "already exists" in stderr
   - Raise RuntimeError on unexpected failure
   - This is idempotent -- safe to call multiple times

4. **extract_pdf_to_markdown(pdf_path: Path) -> tuple[str, int] function:**
   - Use pymupdf4llm.to_markdown(str(pdf_path), page_chunks=True)
   - Join chunk texts with double newlines
   - Return (full_text, page_count)
   - Handle errors gracefully (print [ERROR] and return empty string, 0)

5. **seed_knowledge_base() function:**
   - Set GOOGLE_APPLICATION_CREDENTIALS env var to CREDENTIALS_PATH
   - Call ensure_firestore_db()
   - Create firestore.Client(project=GCP_PROJECT)
   - For each doc_id, pdf_name in PDF_MAP:
     - Check if PDF exists, print [SKIP] if not
     - Extract markdown with extract_pdf_to_markdown
     - Get keywords from KEYWORDS dict (default empty list)
     - Write document with doc_ref.set() (NOT merge -- full overwrite for idempotency):
       - content: full markdown text
       - source_file: pdf_name
       - page_count: int
       - extracted_at: firestore.SERVER_TIMESTAMP
       - version: "1.0"
       - keywords: list from KEYWORDS dict
     - Print [OK] with doc_id, page count, and content length

6. **CLI entry:** `if __name__ == "__main__": seed_knowledge_base()`

7. **Script must be runnable from project root:** `python scripts/seed_firestore.py`
   - All paths relative to project root (KB_DIR = Path("knowledge-base"))
   - The script will ultimately run on VPS #1 where gcloud is available, but the code is written and committed locally

Important: Do NOT use async -- this is a one-shot CLI script. Synchronous Firestore client is fine.
